Bab 8: Bahasa Pemrograman

Rangkuman materi Bahasa Pemrograman

A. Pengertian Pemrograman

Pemrograman adalah proses menulis, menyimpan, dan menjalankan instruksi untuk memberi tahu komputer apa yang harus dilakukan. Programmer menciptakan program untuk mengontrol komputer agar menghasilkan keluaran tertentu. Sejarah pemrograman dimulai dari penemuan Difference Engine oleh Charles Babbage pada tahun 1822 hingga kini berkembang dengan teknologi digital.

B. Fungsi dan Tujuan Program

  • Mengatur kinerja perangkat keras.
  • Mengelola hubungan antar perangkat keras melalui sistem operasi.
  • Menyediakan akses ke berkas dan program lain.
  • Menerjemahkan kode ke bahasa mesin melalui compiler.

C. Jenis-jenis Pemrograman

  • Pemrograman Web: Untuk membuat situs web interaktif (HTML, CSS, PHP, Javascript).
  • Pemrograman Aplikasi: Fokus pada solusi tertentu (Word, Excel, Photoshop).
  • Pemrograman Sistem Operasi: Menjalankan perangkat (Windows, Linux).
  • Pemrograman Game: Menggabungkan desain visual dan algoritma (Unity, Kodu).

D. Bahasa Pemrograman

Bahasa pemrograman adalah aturan sintaks dan semantik untuk membuat program. Programmer menggunakan bahasa ini untuk merancang instruksi yang dapat diproses oleh komputer.

E. Tingkat Bahasa Pemrograman

  • Tingkat Rendah: Bahasa mesin (binary).
  • Tingkat Menengah: Bahasa assembly (ADD, SUB).
  • Tingkat Tinggi: Bahasa yang mudah dimengerti manusia (Python, Java).

F. Contoh Bahasa Pemrograman

  • PHP: Untuk web (WordPress, Facebook).
  • Javascript: Untuk web interaktif.
  • HTML: Membuat struktur website.
  • Python: Populer untuk pemula dan pengembangan aplikasi.
  • C dan C++: Berbasis prosedural dan objek.

G. Rangkuman

Program adalah instruksi yang mengatur komputer agar berfungsi sesuai kebutuhan pengguna. Bahasa pemrograman memudahkan programmer untuk membuat sistem, aplikasi, atau alat lainnya.
